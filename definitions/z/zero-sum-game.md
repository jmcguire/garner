# zero-sum game

zero-sum game \(= a situation in which a gain for one side causes an equal and opposite loss for the other\), an economic phrase dating from the early 1940s, is so written. Some writers, misunderstanding the game theory underlying the phrase, and therefore the phrase itself, mistakenly write \**zero-sum gain*—a malapropism.

 - Current ratio in print \(*zero-sum game* vs. \**zero-sum gain*\): 206:1

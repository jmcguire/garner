# numskull

numskull (= a dolt or dunce) was long the standard spelling (from about 1770 till 2000). *Numbskull*, though an etymologically accurate spelling, was long considered a variant. But it has become far more frequent in print sources since 2000 and vies closely for the dunce cap.

 - Current ratio in print (*numskull* vs. *numbskull*): 1.03:1

# mitosis

mitosis; **meiosis**. Both these terms refer to the process by which a living organism's cells divide. In *mitosis*, a cell's DNA is duplicated; then the cell divides into two separate, genetically identical cells, each with two sets of chromosomes. In *meiosis*, the first stage is the same as mitosis, but the two cells divide again without duplicating the DNA. The four resulting cells each have only one set of chromosomes and are genetically diverse from one another.

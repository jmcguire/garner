# Mephistophelean

Mephistophelean; **Mephistophelian**. The first, which is not quite as common (but nearly\!), should be preferred for the same reasons that *Shakespearean* is preferred. It is pronounced /mә-**fis**-tә-**fee**-lee-әn/ or /**mef**-is-tә-**fee**-lee-әn/. See **Euclidean** & **Shakespearean**.

 - Current ratio in print (*Mephistophelian* vs. *Mephistophelean*): 1.2:1

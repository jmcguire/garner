# processor

processor \(= \[1\] a machine or person that prepares materials for sale, use, or consumption; or \[2\] the central computer component dealing with commands and other information received\), a noun dating only from the early 20th century, has always been predominantly so spelled. \**Processer* is an infrequent variant. See -er \(a\).

 - Current ratio in print \(*processor* vs. \**processer*\): 250:1

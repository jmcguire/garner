# technique

technique; **\*technic**. *Technique* has always been standard in AmE and BrE alike. *\*Technic* is a variant spelling to be avoided—e.g.: "I have heard this remedy \[eye-muscle exercises\] praised by those whom it has helped \(the novelist Aldous Huxley has written a glowing and laudatory book about this *technic* \[read *technique*\]\)." Norman Lewis, *Better English* 238 \(rev. ed. 1961\).

Current ratio in print \(*technique* vs. \**technic*\): 509:1

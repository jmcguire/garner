# track lighting

track lighting (= a system of adjustable lights that can be positioned along an electrified metal track), a phrase dating from the 1940s, is so spelled. Since the 1970s, writers have sometimes erred by writing \**tract lighting*. See **track**. Cf. the *duct*/*duck* issue in **duct tape**.

 - Current ratio in print (*track lighting* vs. \**tract lighting*): 204:1

# Tennessean

Tennessean; \***Tennesseean**. The first has been standard since about 1910, before which time the second (now but a variant) was somewhat more common. Although the final *e* is dropped in this word, it's retained in the similar term *Tallahasseean*. See demonyms.

 - Current ratio in print (*Tennessean* vs. \**Tennesseean*): 12:1

# shortcut > shortcut > shortcut

shortcut > shortcut > shortcut. So inflected. The erroneous past-tense and past-participial *\*shortcutted*, which first emerged in the 1920s, sometimes appears—e.g.: "Calyx and Corolla *short-cutted* \[read *shortcut*\] the route by contracting a network of growers." "Flower Catalog Cuts Middleman," *Milwaukee J. Sentinel*, 14 Jan. 1996, at 12. See irregular verbs.

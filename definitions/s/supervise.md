# supervise

supervise is occasionally misspelled *\*supervize*—e.g.: "And when any out-of-state parolee is under New Hampshire's jurisdiction, it is the Granite State's parole board which *supervizes* \[read *supervises*\] parole." Pat Grossmith, "Humphrey's Criminal Record Wasn't Shared Between Region's States," *Union Leader* (Manchester, N.H.), 19 Oct. 1994, at 4.

 - Current ratio in print (*supervise* vs. \**supervize*): 1,452:1

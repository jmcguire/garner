# skulduggery

skulduggery; **skullduggery**; **\*sculduggery**; **\*scullduggery**. *Skulduggery* \(= trickery; unscrupulous behavior\) has traditionally been the standard spelling in AmE and BrE alike. It remains so in BrE. But in the mid-1990s, about the time of the Clinton–Lewinsky scandal, AmE sources began predominantly using *skullduggery*—and this predominance has continued. It's in keeping with similar AmE spellings such as *skillful* and *willful*, for both of which BrE drops the first *l*. The other two are variants.

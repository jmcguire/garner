# stagy

stagy \(= artificially theatrical or histrionic\), an adjective dating from the early 1800s, has been predominantly so spelled in AmE only since 1991. Before that, *stagy* vied closely in AmE sources with *stagey*, which has always predominated in BrE. The ratios below show that the spelling of this word remains unsettled. Cf. **cagey**.

 - Current ratio in print \(*stagy* vs. *stagey* in AmE\): 1:1.03
 - Current ratio in print \(*stagey* vs. *stagy* in BrE\): 2.4:1

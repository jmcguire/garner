# indictor

indictor \(= someone who indicts or formally accuses\), an agent noun dating from the 1200s, has almost always, in Modern English, been predominantly so spelled—overwhelmingly since 1975. Though given first position in many English-language dictionaries, \**indicter* is but a variant. See -er \(a\).

Current ratio in print \(*indictor* vs. \**indicter*\): 7:1

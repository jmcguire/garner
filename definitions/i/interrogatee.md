# interrogatee

interrogatee; **\*interrogee**. *W3* lists *\*interrogee* \(= someone interrogated\), not *interrogatee*, but the *OED* lists *interrogatee*, not *\*interrogee*. Since the agent noun is *interrogator*, it makes more sense to prefer the corresponding passive form, *interrogatee*.

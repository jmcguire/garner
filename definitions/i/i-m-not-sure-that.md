# I'm not sure that

I'm not sure that; **I'm not sure whether**. The first (more common) phrase means "I doubt" \<I'm not sure that we can make it in time\>; the second means "I wonder if; I don't know if" \<I'm not sure whether Shakespeare died in the 16th or the 17th century\>. (He died in the 17th—in 1616.) Cf. **doubt (a)**.

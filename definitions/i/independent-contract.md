# independent contractor

independent contractor; **employee**. Unlike an *employee*, an *independent contractor* is left free to do the assigned work and to choose the method for accomplishing it. And unlike an *employee*, an *independent contractor* does not, upon committing a wrong while carrying out the work, create liability for an employer that did not authorize the wrongful acts. For example, a freelance writer is an independent contractor; a staff writer is an employee. See **coemployee**.

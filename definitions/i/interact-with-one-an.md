# interact with one another

interact with one another. This is a glaring redundancy that should almost always be trimmed to *interact*—e.g.: "It's the best thing to do in the long run, especially as patients *interact with one another* \[read *interact*\] and form bonds." Ronell Smith, "Families Find Adult Day Care a Comfort," *Augusta Chron.*, 27 Jan. 2003, at B5. The same is true of *interact with each other*. See **each other**.

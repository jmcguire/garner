# if need be

if need be. This phrase, common since the 1600s, can lead to problems of tense-shifting. For example, in the sentence "We always did what we had to do, if need be," the tense abruptly switches from past (*did*) to present (*be*). Eric Partridge suggested *if need were* as a substitute (*U&A* at 150), but that phrase is both archaic and unidiomatic. The best course is often to rephrase with *if necessary*—though in the example above, the phrase is redundant and could be dropped without any loss of meaning. See tenses.

# doughnut

doughnut; **donut**. The first spelling, which is more common, is preferred because it retains the name of the main ingredient \(though on this rationale it might be aptly renamed *sugarnut* or *oilnut*\). *Donut*—or, worse, *do-nut*—should be reserved for eatery names and advertising.

Current ratio in print \(*doughnut* vs. *donut*\): 1.1:1

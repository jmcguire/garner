# differentia

differentia (= a distinguishing characteristic), a Latin loanword dating from the 1500s, has predominantly made the plural *differentiae* /dif-ә-**rent**-shi-ee/ since the mid-1800s—though *differentias* was in frequent use before that time. See plurals (b).

 - Current ratio in print (*differentiae* vs. *differentias*): 23:1

# lacrimation

lacrimation \(= the secretion or shedding of tears, esp. when excessive or abnormal\), a Latin loanword dating in English from the late 1500s, has been predominantly so spelled only since about 1930. Before that, the usual spelling was \**lachrymation*, which is now but a variant.

 - Current ratio in print \(*lacrimation* vs. *\*lachrymation*\): 17:1

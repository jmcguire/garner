# cementitious

cementitious (= of, relating to, involving, or resembling cement), an adjective dating from the mid-1700s, has always been predominantly so spelled. \**Cementious* is a variant. Some people in the construction trades have been known to mispronounce the word as if it were \**cementuous*.

 - Current ratio in print (*cementitious* vs. \**cementious*): 275:1

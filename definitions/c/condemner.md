# condemner

condemner; **condemnor**. The *-er* spelling (/kәn-**dem**-әr/) is preferred in the general sense "one that disapproves." But in AmE, *condemnor* (/kon-dem-**nor**/ or /kәn-**dem**-nәr/) predominates in the sense "a public or semipublic entity that expropriates private property for public use." See -er (a).

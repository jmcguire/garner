# conferrable

conferrable. So spelled. It is pronounced with the stress on the second syllable: /kәn-**fәr**-ә-bәl/. Throughout the 1800s, the spelling *\*conferrible* was common—but that was superseded by 1900. Cf. **referable**.

 - Current ratio in print \(*conferrable* vs. \**conferrible*\): 33:1

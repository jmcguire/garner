# curse > cursed > cursed

curse > cursed > cursed. So inflected. The past-participial adjective *cursed* (= \[1\] under a curse; blasted with a curse; \[2\] heinously wicked; execrable; or \[3\] abominable or damned) has always been predominantly so written in Modern English—except in set phrases such as *curst with age* (as in Robert Burns's line "They zig-zag on, / Till curst with age, obscure an' starvin', / They often groan" \["Epistle to James Smith," 1786\]).

*Cursed* is generally pronounced as one syllable, but it is one of several participial adjectives, like *blessed* and *beloved*, whose older two-syllable versions have also been preserved through use in liturgy and metered verse. In those settings, a grave accent (*cursèd*) is sometimes used as a pronunciation cue. As Hamlet says, "O cursèd spite / That ever I was born to set it right." Shakespeare, *Hamlet*, 1.5.186–87.

 - Current ratio in print (*cursed* vs. *curst*): 121:1

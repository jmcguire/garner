# chasten

chasten; **chastise**. These words are close in meaning, but distinct. *Chasten* = to discipline, restrain, punish, subdue. *Chastise* = (1) to punish, thrash; or (2) to castigate, criticize.

*Chastise* is so spelled; *\*chastize* is incorrect but not uncommon. See -ize.

 - Current ratio in print (*chastise* vs. *\*chastize*): 134:1

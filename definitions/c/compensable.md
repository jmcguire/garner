# compensable

compensable; \***compensatable; \*compensible**. The adjective *compensable* correlates to the noun *compensation*. *\*Compensible* is a fairly common misspelling. No major dictionary records the needless variant *\*compensatable*, which since 1913 has infrequently displaced *compensable*. See -able \(a\); -atable.

Current ratio in print \(*compensable* vs. \**compensatable* vs. \**compensible*\): 1,481:29:1

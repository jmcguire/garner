# catch-22

catch-22; **Hobson's choice**. Not the same in good usage. *Catch-22* = an insoluble problem in which one must do something to do a second thing, which must be done before one can do the first thing. *Hobson's choice* = (1) a situation in which one is presented with only one "choice," and hence no choice at all; or (2) a situation in which either of one's two choices is bad. See **Hobson's choice**.

*Catch-22* derives from the title of Joseph Heller's 1961 novel, in which by rule, you can't fly missions if you're crazy, but if you don't want to fly missions you're not crazy. Today the phrase is not treated as a proper noun, so the standard form is lowercase. Incidentally, Heller originally called his novel *Catch-18* but had the good sense to add four before publication.

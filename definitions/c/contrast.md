# contrast

contrast.

**A. Prepositions with.** One *contrasts* something *with*—not *to*—something else. But it's permissible to write either *in contrast to* or *in contrast with*. Cf. **compare with**.

**B. *Compare and contrast*.** This is an English teacher's redundancy.

**C. Pronunciation.** As a noun, *contrast* is accented on the first syllable: /**kon**-trast/. As a verb, it's accented on the second syllable: /kәn-**trast**/.

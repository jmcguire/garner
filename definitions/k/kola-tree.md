# kola tree

kola tree (the African tree cultivated in various tropical climes for its bitter chestnut-sized nuts), a phrase dating in English from the mid-1800s, has always been predominantly so spelled. \**Cola tree* is a variant.

 - Current ratio in print (*kola tree* vs. \**cola tree*): 1.8:1
 - Current ratio in print (*kola nut* vs. *\*cola nut*): 7:1

# asbestos

asbestos (= a fire-resistant gray mineral formerly used as a building material or in protective clothing but no longer so used because it is now known to be carcinogenic), a term dating from the early 1600s in something like this sense, has the archaic variant spelling \**asbestus*, which mimics the preferred pronunciation: /as-**bes**-tәs/, not /as-**bes**-tohs/.

 - Current ratio in print (*asbestos* vs. \**asbestus*): 1,009:1

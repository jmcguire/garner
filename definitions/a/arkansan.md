# Arkansan

Arkansan; **Arkie**; **\*Arkansawyer**. The first is standard. Some Arkansas denizens don't like it because of the *Kansan* it contains and therefore prefer the fairly common casualism *Arkie*. *\*Arkansawyer* is an infrequent variant. See demonyms.

Current ratio in print \(*Arkie* vs. *Arkansan* vs. \**Arkansawyer*\): 12:9:1

# every so often

every so often; **ever so often**. The first, by far the more common idiom since the early 20th century, means "occasionally" <we go to Lockhart for barbecue every so often>. The second—a quaint phrasing in the manner of *ever so grateful*—means "with great frequency" <you're extremely thoughtful to call me ever so often>. On the dialectal use of *ever so often*, see **ever (b)**.

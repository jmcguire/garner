# easy

easy; **easily**. The word *easy*, generally an adjective, functions arguably as an adverb in some set phrases: *go easy* (*on me*, etc.), *easy does it*, and *take it easy*. But because the verbs can be understood as elliptical linking verbs, the word *easy* can quite appropriately be considered an adjective. Wherever an adverb is needed, of course, *easily* is the right word. See **easier**.

# \*return back

\*return back is a fairly common redundancy. Its heyday was the 1700s through the mid-1800s, after which period it was largely stifled in print sources. But it still occasionally crops up—e.g.: "An initial examination by orthopedic specialist Frank Jobe had shown that Jaha might be able to *return back* \[read *return*\] shortly after the all-star break." Andrew Cohen, "Powerless: Jaha Out for Season," *Wis. State J.*, 18 June 1997, at B1. Cf. \***retreat back** & **revert**.

 - Current ratio in print \(*soon returned to* vs. *\*soon returned back to*\): 218:1

# fontanelle

fontanelle \(= one of several soft spots in a baby's skull where bone has not yet formed and hardened\) has been the predominant spelling in both AmE and BrE since before 1800. Although some dictionaries \(such as *W11*\) list the variant spelling *\*fontanel* as the main headword, it remains but a variant.

Current ratio in print \(*fontanelle* vs. \**fontanel*\): 1.8:1

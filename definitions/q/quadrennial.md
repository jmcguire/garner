# quadrennial

quadrennial; \***quadriennial**. *Quadrennial* (= occurring every four years; lasting four years) is now standard: the etymologically proper *i* was dropped from the word in the 1700s. Today *\*quadriennial* is merely a variant spelling to be avoided. See quadri-.

 - Current ratio in print (*quadrennial* vs. \**quadriennial*): 47:1
